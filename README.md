# @obukata/github-markdown-ckeditor

github-markdown-css 拡張プラグインです。
CKEditor で出力された記事コンテンツ向けにスタイルを統一するためのカスタム調整が含まれています。

---

## 特徴

- github-markdown-css の拡張
- CKEditor で生成された記事コンテンツに最適化


## インストール

```bash
npm i -D @obukata/github-markdown-ckeditor
```

`.node-module/@obukata/github-markdown-ckeditor/styles/github-markdown-ckediro.css`

上記ファイルをHTML上で読み込むだけで CKEditor 用スタイルが適用されます。

## 注意点

* 他のプラグインやユーティリティとの競合に注意してください
